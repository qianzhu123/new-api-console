使用说明 / 安全说明
  1. 打开已登录的 new-api 或 sub2api 网站页面。
  2. 点击扩展图标，再点“采集当前页”。
  3. 确认识别结果后复制 JSON。
  4. 回到本地 qiandao 页面，粘贴到“JSON 导入添加”。

本扩展会读取当前站点 cookies，包括 HttpOnly cookie，所以可以拿到普通油猴脚本无法读取的 new-api session。请只在自己的账号上使用，不要把导出的 JSON 发给别人。
